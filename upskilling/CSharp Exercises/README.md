C# Exercises
==============

This folder contains a single .NET console project with concise example implementations for the requested C# exercises (1–30).

Build & run
-----------

Requires .NET SDK (7.0 or later).

Open a terminal in this folder and run:

```bash
dotnet run
```

Pick an exercise number when prompted. Exercise 30 (ADO.NET) contains a template that needs a valid connection string and package references to actually run against SQL Server.

Notes
-----
- The project is minimal and focuses on illustrative code. For ADO.NET and JSON/file operations you may need to run the program where the runtime has permission to write files.
- For C# 12-specific features (primary constructors, required modifier) the code uses compatible constructs; using preview features may require a newer compiler.
