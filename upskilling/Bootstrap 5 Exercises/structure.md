# Bootstrap Directory Structure (exercise 2.1)

- `css/` - contains compiled CSS files (e.g., `bootstrap.min.css`). Used for styling and utility classes.
- `js/` - contains compiled JavaScript bundles (e.g., `bootstrap.bundle.min.js`) which include Bootstrap's plugins (collapse, modal, dropdown) and Popper.
- `icons/` - contains Bootstrap Icons (SVG or font files) for easy vector icons.

Purpose:
- CSS: layout, grid, components, utilities.
- JS: interactive components that require JavaScript (modals, tooltips, accordion, dropdowns).
- Icons: optional visual icons used in navigation, buttons, and UIs.

In this folder `vendor/bootstrap` is a placeholder demonstrating the structure.
